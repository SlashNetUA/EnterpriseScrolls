# Function 13h: Delete File

## Parameters

**DE** = Pointer to unopened FCB

## Results

**L**=**A** =
 - **00h**: if files deleted OK
 - **FFh**: if no files deleted

## Description

All files in the current directory of the disk specified by the FCH, and which match the ambiguous filename in the FCB, are deleted. Sub-directories, system files and read-only files are not deleted. If any files at all are successfully deleted then this function returns with **A**=**0**. A return with **A**=**FFh** indicates that no files were deleted.
