# Function 05h: List Output

## Parameters

**E** = Character to be output

## Results

None

## Description

The character passed in register **E** will be sent to the printer. The printer channel is opened by default to the '**PRINTER:**' device but can be altered by the [MODE](../../dos-commands/cmd-mode.md) command. This allows a serial printer to be used, and can also be used to send printer output to a file. The same channel is used for console output which is echoed to the printer, and in this case TABs will he expanded to eight columns.
