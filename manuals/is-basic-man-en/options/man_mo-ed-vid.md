# EDITOR VIDEO

`SET EDITOR VIDEO channel-number`  
`ASK EDITOR VIDEO var`

Allows the specified channel to be used as the text page for the editor, for use with editor channels subsequently opened. Can be used with [ASK](../commands/man_cs-ask.md).
