# REM1 / REM2

`SET REM1 ON/OFF`  
`TOGGLE REM1`  
`ASK REM1 var`

`SET REM2 ON/OFF`  
`TOGGLE REM2`  
`ASK REM2 var`

Controls remote control switches 1 and 2. (Also controlled by tape operations.)

|     Setting      | Status  | Variable value<br>(after ASK) |
|:----------------:|:-------:|:-----------------------------:|
|        ON        | release |               0               |
| OFF<br>(default) |  hold   |              255<br>(≠0)              |
