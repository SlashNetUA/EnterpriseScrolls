# EDITOR BUFFER

`SET EDITOR BUFFER expr`  
`ASK EDITOR BUFFER var`

Defines the size of the editor's buffer, in 256-byte chunks, for use with editor channels subsequently opened. Can be used with [ASK](../commands/man_cs-ask.md).
