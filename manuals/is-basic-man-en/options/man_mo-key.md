# KEY CLICK

`SET KEY CLICK ON/OFF`  
`TOGGLE KEY CLICK`

Determines whether a click is heard with each key-press. Can be used with [TOGGLE](commands/man_cs-toggle.md).

See also [EXOS VAR 7](../../../programming/system-info/exos-variables/exos_var7.md).   

# KEY DELAY

`SET KEY DELAY expr`  
`ASK KEY DELAY var`

Sets the initial keyboard delay before auto-repeat starts, in units of **1/50** second. Can be used with [ASK](commands/man_cs-ask.md). Default value - **30**.  

See also [EXOS VAR 11](../../../programming/system-info/exos-variables/exos_var11.md).   

# KEY RATE

`SET KEY RATE expr`  
`ASK KEY RATE var`
 
Specifies the keyboard auto-repeal rate, in units of **1/50** second. Can be used with [ASK](commands/man_cs-ask.md). Default value - **3**.

See also [EXOS VAR 10](../../../programming/system-info/exos-variables/exos_var10.md).
