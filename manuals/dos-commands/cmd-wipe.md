# WIPE (transient)

## Applicable systems

**IS-DOS**: all versions

## Format

`WIPE`

## Purpose

Clear function keys.

## Use

Clears previously set function keys except **F8**, which remains [PRINTER](cmd-printer.md).

Note that this is a transient command, and must be loaded from disk.

## Example

`WIPE`

---- 

See also [FKEY](cmd-fkey.md).
