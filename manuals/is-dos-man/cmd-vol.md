# VOL

## Format

`VOL [d:] [filename]`

## Purpose

Displays or changes the volume name on a disk.

## Use

If  no parameters are given,  or if only  a  drive name  is given,  then the volume name of the current or specified  drive is printed.  If a filename  is  given, then the volume name of the default or current drive is changed to the specified filename.  Note that specified filename may contain characters not normally allowed in filenames, such as spaces.

## Examples

`VOL B:`  
_Volume in drive B: has no name_  

`VOL B:BACKUP`
