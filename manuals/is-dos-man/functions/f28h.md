# Function 28h: Random Write with Zero Fill

## Parameters

**DE** = Pointer to opened FCB

## Results

**L**=**A** =
 - **00h**: if no error
 - **01h**: if error

## Description

This function is identical to '[RANDOM WRITE](f22h.md)', except that if the file has to be extended, any extra allocated disk clusters will be filled with zeroes before writing the data.
