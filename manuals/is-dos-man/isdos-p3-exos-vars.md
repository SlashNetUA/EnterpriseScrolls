# 6. EXOS System Variables

The following is a list of the EXOS system variables which can be controlled by IS-DOS.

|        Number        | Name                                                                                                                                              | Function                                                                                    |
|:--------------------:| ------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
|          64          | [ROM_EXDOS](../../programming/system-info/exos-variables/exos_var64.md)                                                                           | Segment number of the EXDOS ROM                                                             |
| 65<br>66<br>67<br>68 | [IS_P0<br>IS_P1<br>IS_P2<br>IS_P3](../../programming/system-info/exos-variables/exos_var65-68.md)                                                 | Disk transfer segments for paging mode 2                                                    |
|          69          | [ECHO](../../programming/system-info/exos-variables/exos_var69.md)                                                                                | ECHO flag for IS-DOS batch files                                                            |
|          70          | [VERIFY](../../programming/system-info/exos-variables/exos_var70.md)                                                                              | EXDOS verify flag (**0** = **on**, **FFh** = **off**)                                       |
|          71          | [DEF_UNIT](../../programming/system-info/exos-variables/exos_var71.md)                                                                            | Default unit (drive) number (**1**...**26**)                                                |
|          72          | [BOOT_DRV](../../programming/system-info/exos-variables/exos_var72.md)                                                                            | Drive for booting IS-DOS                                                                    |
|          73          | [STEP_RATE](../../programming/system-info/exos-variables/exos_var73.md)                                                                           | Step rate for UNITH (**0** = **6ms**, **1** = **12ms**, **2** = **20ms**, **3** = **30ms**) |
|          74          | [DSK_CHK](../../programming/system-info/exos-variables/exos_var74.md)                                                                             | Zero to enable full disk change checking for already open files                             |
|       75<br>76       | [IN_ERROR](../../programming/system-info/exos-variables/exos_var75.md)<br>[OUT_ERROR](../../programming/system-info/exos-variables/exos_var76.md) | Input and output channels for EXDOS default error handling                                  |
|       77<br>78       | [IN_CLI](../../programming/system-info/exos-variables/exos_var77.md)<br>[OUT_CLI](../../programming/system-info/exos-variables/exos_var78.md)     | Input and output channels for EXDOS CLI commands                                            |
|          79          | [DT_FORM](../../programming/system-info/exos-variables/exos_var79.md)                                                                             | EXDOS CLI date and time format                                                              |
|          80          | [ER_MESS](../../programming/system-info/exos-variables/exos_var80.md)                                                                             | Explain error disable flag                                                                  |
|          81          | [AB_ERR](../../programming/system-info/exos-variables/exos_var81.md)                                                                              | Disk error code for .ABORT error                                                            |
|          82          | [DSK_ERR](../../programming/system-info/exos-variables/exos_var82.md)                                                                             | Error from last IS-DOS command                                                              |
|          83          | [CLI_PROT](../../programming/system-info/exos-variables/exos_var83.md)                                                                            | Protection flag for IS-DOS CLI                                                              |
| 84<br>85<br>86<br>87 | [RND_0<br>RND_1<br>RND_2<br>RND_3](../../programming/system-info/exos-variables/exos_var84-87.md)                                                 | 32 bit random number maintained by DISKIO for volume ids                                    |
|          88          | [CLI_EN](../../programming/system-info/exos-variables/exos_var88.md)                                                                              | Non-zero to disable EXDOS CLI                                                               |
|          89          | [DCH_DIS](../../programming/system-info/exos-variables/exos_var89.md)                                                                             | Non-zero to make EXDOS believe 'disk not change' responses from unit handlers               |
|          90          | [FAST_VID](../../programming/system-info/exos-variables/exos_var90.md)                                                                            | Zero to enable IS-DOS fast video by default                                                 |
|          91          | [CD_PROMPT](../../programming/system-info/exos-variables/exos_var91.md)                                                                           | Zero to enable printing of current directory as the IS-DOS prompt                           |
