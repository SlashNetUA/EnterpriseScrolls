# STATUS

`SET STATUS ON/OFF`  
`ASK STATUS var`  
`TOGGLE STATUS`

Turns the '**status line**' (at the top of the display) **on** or **off**. Can be used with [TOGGLE](man_cs-toggle.md).

**0**: Status on
