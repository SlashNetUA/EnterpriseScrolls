# GOTOXY(X,Y)

(тільки у версії 1.2)

Процедура встановлює курсор у позицію символу з координатами (X,Y) на поточному каналі. Координата лівого верхнього кута екрана (1,1); горизонтальні позиції збільшуються зліва направо, а вертикальні – зверху вниз. X і Y — integer.


Процедура встановлює курсор у позицію символу з координатоми **X**, **Y** на поточному каналі. Координати лівого верхнього кута екрана — **(1, 1)**; значення горизонтальної позиції **X** зростають зліва направо, а вертикальної **Y** — зверху вниз. Якщо якесь із значень дорівнює **0**, курсор залишається в тому самому стовпці / рядку. Змінні **X** та **Y** є виразами типу **INTEGER**.

Хоча серед процедур _Hisoft Pascal_ немає процедури `TAB(X)` (яка виконує табуляцію курсора), її можна замінити процедурою `GOTOXY`. Адже якщо як номер рядка або стовпця передати нуль, курсор залишиться в поточному рядку / стовпці:

```
PROGRAM PASCAL;
CONST SOR=12;
VAR C,I,K:INTEGER;
BEGIN
	CLRSCR;
	FOR I:=0 TO SOR DO BEGIN
		GOTOXY(37-I*3,0);
		C:=1;
		FOR K:=0 TO I DO BEGIN
			WRITE(C:6);
			C:=C*(I-K) DIV (K+1)
		END;
		WRITELN
	END
END.
```
