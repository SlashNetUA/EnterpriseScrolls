# Керуючі символи
## Для усіх відеосторінок
`^Z` (**1Ah**): очистити екран та встановити курсор/промінь на початкову позицію.  
`^J` (**0Ah**):  Line-feed. Перемістити курсор донизу на наступний рядок (якщо курсор внизу текстового екрану то здійснюється прокрутка екрану за умови що прокручування активне)  
`^M` (**0Dh**): Carriage return. Повертає курсор на початок поточного рядка.  
`^^` (**1Eh**): Встановити курсор/промінь на початкову позицію (ASCII RS).  

`escK`: Призначити символ *(див. нижче)*.  
`escС`: Встановити усі кольори палітри *(див. нижче)*.  
`escс`: Встановити колір палітри *(див. нижче)*.  
`escI`: Вибрати колір "пензля" *(див. нижче)*.  
`escP`: Вибрати колір "паперу" *(див. нижче)*.  
`esc=`: Встановити курсор на координати *(див. нижче)*.  

## Для текстових відеосторінок
`^Y` (**19h**): Очистити рядок від курсора до кінця рядка.  
`^H` (**08h**): Перемістити курсор вліво (ASCII BS).  
`^I` (**09h**): Перемістити курсор вправо (ASCII TAB).  
`^K` (**0Bh**): Перемістити курсор догори (ASCII VT).  
`^V` (**16h**): Перемістити курсор униз (ASCII SYN).  

`esc?`: Отримати поточні координати курсора *(див. нижче)*.  
`esc.`: Змінити символ курсора на інший *(див. нижче)*.  
`escM`: Змінити колір курсора *(див. нижче)*.  
`escO`: Відобразити курсор.  
`esco`: Приховати курсор.  
`escS`: Дозволити автоматичну прокрутку екрану.  
`escs`: Заборонити автоматичну прокрутку екрану.  
`escU`: Прокрутити екран догори *(див. нижче)*.  
`escD`: Прокрутити екран донизу *(див. нижче)*.  

## Для графічних відеосторінок
`escA`: Встановити промінь на координати *(див. нижче)*.  
`escR`: Перемістити промінь *(див. нижче)*.  
`esc@`: Отримати поточні координати променю *(див. нижче)*.  
`escS`: Увімкнути промінь.  
`escs`: Вимкнути промінь.  
`esc.`: Змінити Line style *(див. нижче)*.  
`escM`: Змінити Line mode *(див. нижче)*.  
`esca`: Змінити тип малювання у атрибутному режимі *(див. нижче)*.  
`escF`: Заповнити замкнений контур кольором *(див. нижче)*.  
`escE`: Намалювати еліпс *(див. нижче)*.  

# Escape-послідовності

Одним із варіантів введення послідовностей у конкретний канал є команда [PRINT](../is-basic_man-en/man_cs-print.md) Бейсіка:  
```
PRINT #ch:CHR$(27);"o";
```
наприклад, цей рядок приховає курсор на конкретній відеосторінці. 

esc**=** [Position cursor](esc-seq/1b3d-position-cursor.md)  
esc**C** [Set Palette Colours](esc-seq/1b43-palette-colours.md)  
esc**с** [Change Palette Colour](esc-seq/1b63-palette-colour.md)  
esc**K** [Define character](esc-seq/1b4b-define-character.md)  


**6Fh** set cursor off
