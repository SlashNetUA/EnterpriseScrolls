# Зміна коду символа для відмальовки курсора (текстові відеосторінки)

`esc.<n>`

hex: `1B`,`2E`,**n**  

dec: `27`,`46`,**n**  

Встановити символ курсора на код символу **n**.

Значення по замовчуванню: **142** (**8Eh**). 

Set cursor character to character code **n**.
