# 200 - VDUMP_RES (Enterprise Plus)

`ASK 200 var`  
`SET 200, expr`  
`TOGGLE 200` - inverts value.

Ha a VDUMP utasítás EPSON nyomtatóra dolgozik, akkor e változó értéke határozza meg a nyomtató pixel üzemmódját.

**0**: 480 pixels.  
**1**: 960 pixels.  
**2**: 960 pixels (kiemelt).  
**3**: 1920 pixels.
