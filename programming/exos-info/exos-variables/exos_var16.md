# 16 - BAUD_SER

`ASK 16 var`  
`SET 16, expr`  
`TOGGLE 16` - inverts value.

Defines serial baud rate.

|Parameter value|Baud rate| 
|:-------------:|--------:|
|0|50 baud
|1|75 baud
|2|110 baud
|3|134.5 baud
|4|150 baud
|5|200 baud
|6|300 baud
|7|600 baud
|8|1200 baud
|9|1800 baud
|10|2400 baud
|11|3600 baud
|12|4800 baud
|13|7200 baud
|14|9600 baud
|15<br>(default)|9600 baud

Memory address: **49109**


See also [SET SERIAL BAUD](../../is-basic/man_mo-ser-baud.md)
