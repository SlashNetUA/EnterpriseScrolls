# Конверсія ігор з платформи Pico-8

<div style="text-align:center;">

Pico-8 – це так звана "фентезійна консоль", вигадана 8-бітна система, розроблена Lexaloffle Games як програма для різних ОС. Вона імітує суворі технічні обмеження ретро-комп'ютерів 80-х, щоб стимулювати креативність розробників.

Щоб відтворити атмосферу розробки під старе обладнання, Pico-8 має жорсткі ліміти:
*   **Екран:** 128 x 128 пікселів.
*   **Палітра:** 16 фіксованих кольорів.
*   **Звук:** 4 канали для чиптюну, з вбудованими синтезаторами SFX та трекером.
*   **Код:** Мова програмування Lua.
*   **Розмір гри:** Обмежений обсягом коду (8192 токени) та пам'яттю. Зберігається як .png-картридж.
