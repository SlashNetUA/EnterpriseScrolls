# IS-Basic BAS file format

## Header

| Name             | Length  | Byte offset | Description                                                                |
| ---------------- | ------- |:-----------:| -------------------------------------------------------------------------- |
| EXOS module type | 2 bytes |      0      | `00 03`: IS-BASIC programs in 1 file<br>`00 04`: IS-BASIC program (single) |
|                  |         |             |                                                                            |
