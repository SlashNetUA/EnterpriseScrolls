# z88dk

![](https://raw.githubusercontent.com/wiki/z88dk/z88dk/images/logo.gif)

[Сторінка проекту](https://github.com/z88dk/z88dk)

Z88DK — це повний набір програм для розробки програм для мікропроцесорів 8080, 8085, gbz80, z80, z180, z80n, kc160 та Rabbit.

До його складу входять два компілятори для мови C, а також асемблер, лінкер та керування бібліотеками коду. Крім того, він має інструменти для стиснення даних та утиліти для конвертації бінарного коду у потрібний формат для певних пристроїв.

Створення програм на асемблері або C тут повністю взаємопов'язане: проєкти можна писати цілком на асемблері, цілком на C, або поєднувати ці дві мови. Інструментарій ставиться до обох мов як до рівноцінних і розроблений так, щоб максимально спростити їхнє вільне використання разом. Це означає, що функції, написані на C та асемблері, можуть звертатися одна до одної або використовувати готові, дуже швидкі бібліотечні функції.
