# MACHINE OPTIONS (GENERAL)

Certain system variables and machine functions can be controlled directly from BASIC; these are called machine options. To assign a value to an option, the command [SET](man_cs-set.md) is used. Where stated, the options listed below may also be handled in conjunction with [ASK](man_cs-ask.md) or [TOGGLE](man_cs-toggle.md).
