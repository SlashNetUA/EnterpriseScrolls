# EXOS Channels

Channels numbers range **0** to **254** (**255** is an invalid channel number which is used for special purposes)


## IS-BASIC

See [OPEN](../../manuals/is-basic-man-en/commands/man_cs-open.md) for more info.

**#0**: Default EDITOR channel  
**#1**-**#100**: For using in Basic programs  
**#101**: Default GRAPHICS screen channel  
**#102**: Default TEXT screen channel  
**#103**: Default SOUND channel  
**#104**: Default PRINTER channel  
**#105**: Default KEYBOARD channel  
**#106**: Used for file-based input/output operations  
**#107**: Used for network operations

## HiSoft Pascal

**#103**: Default SOUND channel  
**#105**: Default KEYBOARD channel  
**#120**: Default TEXT screen channel  
**#121**: Default editor channel with 2K buffer  
**#122**-**#123**: Used for TAPE, PRINTER and etc.  
**#131**: Default GRAPHICS screen channel (2-col)  
**#132** and higher: For using in Pascal programs
