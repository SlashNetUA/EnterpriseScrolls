# Определение цветов

Чип Nick генерирует байты цвета для каждого пикселя на экране в зависимости от значений регистров видео, цветового режима и палитры. Эти 8-битные значения цвета преобразуются в отдельные сигналы красного, зелёного и синего, которые подаются на разъём монитора. Оттуда они подаются на PAL-кодер и UHF-модулятор для преобразования в сигнал, подходящий для телевидения.  
Из 8 бит 3-3 определяют красный и зеленый цвета, а 2 - синий цвет, поэтому красный и зеленый могут иметь 8 оттенков, а синий - 4. (G=зеленый, R=красный, B=синий)

|   |   |   |   |   |   |   |   |
|---|---|---|---|---|---|---|---|
|б7|б6|б5|б4|б3|б2|б1|б0|
|Г2|Р2|В1|Г1|Р1|В0|Г0|Р0|

Три бита создают цвета по следующей формуле. B0, G0, R0 — самые значимые биты каждого цвета, синий может иметь только 4 различных значения.

> полностью красный = (R2+R1*2+R0*4)/7  
> всего зеленого = (G2+G1*2+G0*4)/7  
> полностью синий = (B1+B0*2)/3
