# Зміна кольору курсора

Тип каналів: текстові відеосторінки

Формат: 

`escM<n>`

hex: `1B`,`4D`,**n**  

dec: `27`,`77`,**n**  

Змінити колір курсора на колір палітри **n**.

Значення по замовчуванню: **3**. 

Set cursor to palette colour **n**.
