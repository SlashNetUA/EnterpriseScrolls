# LPT (Line Parameters Table)



На відміну від більшості тогочасних відеочипів (як-от у ZX Spectrum чи C64), які автоматично та невидимо для програміста керували кадровим гасінням (VBlank) та кадровим синхроімпульсом, Nick цього сам не робить. Програміст має власноруч додати в кінець таблиці LPT спеціальні «пусті» блоки параметрів (LPB), які не виводять графіку, а натомість апаратно виставляють біти вертикального синхроімпульсу (Vertical Sync) та гасіння, щоб телевізор чи монітор зрозумів, де закінчується один кадр і починається наступний.
