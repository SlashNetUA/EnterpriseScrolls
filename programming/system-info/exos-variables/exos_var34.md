# 34 - PROTECT

`ASK 34 var`  
`SET 34, expr`  
`TOGGLE 34` - inverts value.

Set to zero to make cassette write out protected file.

**255**: (default) non protected  
 **≠0**: non protected  
**0**: protected  

Memory address: **49127**
