# 26 - ST_FLAG

`ASK 26 var`  
`SET 26, expr`  
`TOGGLE 26` - inverts value.

**0**: Status line visible (default).  
**255** (**≠0**): Hidden status line.  
**42**: Show `Written by: Mrl BT NMV GNH CGE AEL` easter egg.
