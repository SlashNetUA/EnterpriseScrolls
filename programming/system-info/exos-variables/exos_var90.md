# 90 FAST_VID

Zero to enable IS-DOS fast video by default

Specifying **ON** enables IS-DOS's internal video driver to be used for screen output in place of the normal EXOS video channel. Screen operations will then be faster, though more memory will be used. The default is **OFF**. See the [Programmer's Reference Section](../../../manuals/is-dos-man-en/functions/f85h.md).
